# 明清中国科技树、思潮与时代设计
# 基于历史事实，不考虑现有游戏科技

## 时代划分（Ages）

### Age 1: 明初重建（1368-1435）
**时代特征**：明朝建立，恢复生产，重建制度
- 洪武之治
- 永乐盛世
- 恢复科举
- 重建农业

### Age 2: 明中繁荣（1435-1572）
**时代特征**：经济繁荣，文化发展，制度完善
- 成化中兴
- 弘治中兴
- 商业繁荣
- 文化昌盛

### Age 3: 明末变革（1572-1644）
**时代特征**：制度改革，军事变革，思想活跃
- 张居正改革
- 万历中兴
- 军事改革
- 思想解放

### Age 4: 清初统一（1644-1735）
**时代特征**：统一全国，制度融合，康乾盛世
- 入关统一
- 制度融合
- 康乾盛世
- 疆域扩张

### Age 5: 清中守成（1735-1840）
**时代特征**：守成时期，文化整理，对外封闭
- 乾嘉考据
- 文化整理
- 闭关锁国
- 人口增长

### Age 6: 清末变革（1840-1912）
**时代特征**：外患内忧，洋务运动，近代化尝试
- 鸦片战争
- 洋务运动
- 戊戌变法
- 清末新政

## 思潮（Ideas/Thoughts）

### 1. 理学（Neo-Confucianism）
**时代**：Age 1-6
**特征**：程朱理学成为官方思想
- 格物致知
- 存天理灭人欲
- 影响科举和教育

### 2. 心学（School of Mind）
**时代**：Age 2-3
**特征**：王阳明心学兴起
- 知行合一
- 致良知
- 思想解放

### 3. 实学（Practical Learning）
**时代**：Age 3-4
**特征**：经世致用，注重实用
- 经世致用
- 实用技术
- 改革思想

### 4. 考据学（Textual Criticism）
**时代**：Age 5
**特征**：乾嘉学派，整理古籍
- 训诂考据
- 古籍整理
- 学术严谨

### 5. 经世致用（Statecraft）
**时代**：Age 3-6
**特征**：关注实际问题，寻求解决方案
- 关注现实
- 改革主张
- 实用主义

### 6. 西学东渐（Western Learning）
**时代**：Age 6
**特征**：学习西方科技和思想
- 学习西方
- 中体西用
- 近代化尝试

## 设计说明

1. **历史准确性**：所有科技都基于真实历史事件和制度
2. **时代特征**：每个时代的科技反映该时期的主要特点
3. **逻辑连贯**：科技之间有合理的前置关系
4. **平衡性**：各条线路都有发展空间
5. **特色**：突出明清中国的独特制度和文化
